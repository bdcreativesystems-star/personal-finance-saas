# Personal Finance SaaS

A SaaS-style Personal Finance Tracker built with Python, Flask, SQLAlchemy, and Chart.js.  
Track your expenses, visualize spending, and manage finances like a professional app.

![Dashboard Screenshot](screenshot.png)

## Features

- User authentication (register/login)  
- Track expenses by category  
- Dashboard with summary cards  
- Bar chart visualization of spending  
- Recent expense table  
- Modern SaaS-style UI  

## Tech Stack

- Python, Flask, SQLAlchemy  
- HTML, CSS, Jinja Templates  
- Chart.js for charts  
- SQLite (development)
